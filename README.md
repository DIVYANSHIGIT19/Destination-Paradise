# Destination Paradise – Travel Booking Website

Destination Paradise is a responsive frontend travel booking website developed using HTML, CSS, JavaScript, and Bootstrap. The project focuses on modern UI design, responsive layouts, and interactive user features.

## Features

• Responsive Navigation Bar with dropdown  
• Dynamic travel packages rendering using JavaScript  
• Live search filtering system  
• Booking form with validation  
• Email and phone number validation  
• Age verification (18+ restriction)  
• Dynamic destinations dropdown  
• Automatic image carousel  
• Mobile responsive design  

## Technologies Used

HTML5  
CSS3  
JavaScript  
Bootstrap 5  

## Project Objective

The objective of this project was to strengthen frontend development skills by implementing real-world UI components, form validation, and DOM manipulation techniques.
